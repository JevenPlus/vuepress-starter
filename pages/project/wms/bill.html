    # 应收账单(折扣/返点)

    # 需求背景

    客户报价折扣需求

    1. 公司根据客户情况，给予部分客户基础运费折扣，需要对基础部分进行折扣处理
    2. 公司更具客户情况，客户达到一定单量，给予基础运费一定折扣

    客户返点需求

    1. 当客户达到一定的单量后，针对基础运费对部分客户给予返点
    2. 当客户月账单达到一定金额后，针对基础运费对部分客户给予返点

    注意

    以上折扣仅针对发货方式为无忧达快递的订单基础运费进行返点，自上传的与第三方快递不包含在折扣范围内。

    # 做什么？

    1. 账单确认页面新加一个折扣输入框(输入限制：90%≤折扣≤100%)和一个确认按钮，输入完成，点击确认后就不能再编辑(禁用该输入框)，没有点击确认前则可以随意修改折扣
    2. 折扣确认完成后基础运费折扣后的金额和折扣金额
    3. 列表中新增显示折扣前金额和折扣后金额两个字段
    4. 主列表中新增显示折扣前金额和折扣后金额两个字段

    # 疑问点

    1. 如果不满足条件，但是输入了折扣，并点击了确认按钮，则该账单下的基础运费按照折扣后的金额进行计费 -- 已确认
